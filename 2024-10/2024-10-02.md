### 엘리스 트랙 20일차
- REST API
  - (직관적으로) 사용자의 요청을 HTTP 프로토콜을 사용해 응답하는 API
  - (개념적으로) HTTP 프로토콜에 기반하여 "REST 아키텍처를 따르는 API"

- REST ?
  - **Re**presentational **S**tate **T**ransfer
  - 자원을 이름으로 구분하여 자원의 상태 정보를 주고 받는다.

- 구성 요소
1. 자원(Resource) - URI
2. 행위(Verb) - HTTP 메서드
3. 표현(Representations) - HTTP 메세지 페이로드
   ex) JSON, XML
