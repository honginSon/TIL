## 
- Unix Directory Structure

- CPU 
  : Processor

- RAM 
  : Memory

- SSD, 하드 디스크 
  : Storage

- Program은 Storage에 저장되어 있는 **명렁어 코드**이다.
- Storage에 있던 Program이 Memory로 올라가서 Processor에 의해 실행되면 Process가 된다.
- 즉, Process는 **실행중인 Program**이다.
